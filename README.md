# FunnyDrop - Joke Collection App

A full-stack joke collection web application built with Node.js, Express.js, and Vanilla JavaScript following the MVC pattern.

## Setup Instructions

### Requirements
- Node.js installed
- Internet connection (for first install)

### How to Run

1. Clone or download the project
2. Open terminal inside the project folder
3. Install dependencies:
   npm install
4. Start the server:
   npm start
5. Open browser and go to:
   http://localhost:3000

## Features
- Register and Login with JWT authentication
- View all jokes
- Add, Edit, Delete your own jokes
- Like jokes
- Search and filter by category

## Tech Stack
- Backend: Node.js + Express.js (MVC pattern)
- Frontend: Vanilla JavaScript
- Database: JSON Server (db.json)
- Authentication: JWT + bcrypt
